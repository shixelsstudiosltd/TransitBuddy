Transits Database (tb_transits)

_idunique transit id
item_idid of item being transited used to fetch all items who have a transit_budy = this users id and get the following information -- tracking number, item type, shipper_id, origin, destination, weight, reward, speed, images, expected delivery, and transit_status
transit_buddyuser_id for user who is transiting the item
